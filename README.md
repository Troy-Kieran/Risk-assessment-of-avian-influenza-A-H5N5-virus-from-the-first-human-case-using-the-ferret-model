# Overview
R code and data for Pulit-Penaloza et. al: 

Figure 2C (JPP_WA2148_10Mar2026.R; H5N5_symptom_prevalence_10Mar2026.csv) and 

Figure 5 C/D (H5N5_AR_context_plot.R; combined_data_8July26.csv), Includes new data from this study as well as data sourced from 
Pulit-Penaloza JA, Kieran TJ, Brock N, Belser JA, Sun X, Zeng H, Pappas C, De La Cruz JA, Hatta Y, Di H, Davis TC, Tumpey TM, Maines TR. (2026). Advancing A(H5N1) influenza risk assessment in ferrets through comparative evaluation of airborne virus shedding patterns. Nature Communications. 17, 2266. https://doi.org/10.1038/s41467-026-68931-1

Additional figures created in GraphPad Prism have source data provided ().
